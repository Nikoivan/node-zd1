# Начало работы

## Установка зависимостей 
  - Выполнить команду "npm install (npm i)" - произойдет необходимая установка требующихся зависимостей проекта.

## Сборка проект 
 - Выполнить команду "npm run build" - произойдет сборка проекта. Появится папка dist с результатом компиляции

## Запуск консольного приложения в режиме разработки 
  - npm run dev -- <command> [options]

### Допустимы команды
- current     — получить текущую дату/время
- add         — получить дату в будущем
- sub         — получить дату в прошлом

### Допустимые опции 
 -  -y / --year

 -  -m / --month

 -   -d / --date

## Запуск консольного приложения
- npm run start -- <command> [options]

- add         — получить дату в будущем
- sub         — получить дату в прошлом

### Допустимые опции
